#function for importing the dataset
def import_dataset(dir_path,label):
    my_dir_path = dir_path
    mail = defaultdict(list)
    if label == 0:
        for file in Path(my_dir_path).iterdir():
            with open(file,"r") as file_open:
                mail["file_name"] = file.name
                mail["text"].append(file_open.read())
    if label == 1:
        for file in Path(my_dir_path).iterdir():
            with open(file,"r",encoding = "latin-1") as file_open:
                mail["file_name"] = file.name
                mail["text"].append(file_open.read())
    df = pd.DataFrame(mail)
    df.drop("file_name", axis=1, inplace=True)
    df.rename(columns={'text': 'mail'}, inplace=True)
    if label == 0:
        df['status'] = 0
    if label == 1:
        df['status'] = 1
    return df

Data Preprocessing¶

Created a function process_message which will take an email and preprocessed it.It will perform the below operations:

  1. Received an email as an argument.
  2. Convert all the text into lowercase.
  3. Tokenization : It involves splitting up a message into pieces and Removing all the punctuation marks.
  4. Removed stopwords from the email.Stopwords are the words which does not give much meaning to a sentence and they can be removed without affecting the meaning of the sentence.
  5. Peform Stemming of the emails.It will cut off the end or the beginning of the word, and will take into consideration a list of common prefix and suffix that were observed in an inflected word.
In [11]:
#function for preprocessing the emails
def process_message(mail):
    clean_mail = " "   
    mail = mail.lower() #convert emails to lowercase
    nopunc =[char for char in mail if char not in string.punctuation]  #remove punctuation marks
    nopunc=''.join(nopunc)
    clean_word = [word for word in nopunc.split() if word.lower() not in stopwords.words('english')] #remove stopwords
    clean_word = [w for w in clean_word if len(w) > 2]
    ps = PorterStemmer() #perform stemming
    clean_word = [ps.stem(word) for word in clean_word]
    for i in range(len(clean_word)):
        clean_mail = clean_mail + " "+ clean_word[i]
    return clean_mail

#create a function to return the 20 topwords from the emails
def top_words(mails):
    words = mails['mail'].tolist()
    token = [word_tokenize(x) for x in words]
    words = reduce(operator.concat,token)
    top_words = dict(Counter(words).most_common(20))
    return top_words

Feature extraction (tf-idf)¶

Tf-idf was used for feature extraction.Tf-idf stands for term frequency-inverse document frequency.The tf-idf weight is a weight used in information retrieval and text mining.The tf-idf weight is composed of two terms: the first computes the normalized Term Frequency (TF) means the number of times a word appears in a document, divided by the total number of words in that document.
TF(t) = (Number of times term t appears in a document) / (Total number of terms in the document)
The second term is the Inverse Document Frequency (IDF), computed as the logarithm of the number of the documents in the corpus divided by the number of documents where the specific term appears.
IDF(t) = log_e(Total number of documents / Number of documents with term t in it)
Tf-idf was applied on training,validation and testing dataset accordingly.

In [31]:
#create function for tuning the hyperParameter,fitting the model on training dataset and predicting on validation dataset
def model_execution(model,hyperParameter,tfidf_training,training_set_status,tfidf_validation):
    classifier_tuned = RandomizedSearchCV(model,hyperParameter)
    classifier_tuned.fit(tfidf_training, training_set_status.values.ravel())
    pred = classifier_tuned.predict(tfidf_validation)
    return pred,classifier_tuned

Next a function model_evaluation was created to evaluate the model.It take validation set and predicted values as arguments.It prints the confusion matrix and return the accuracy for the model.

#create a function to investigate kinds of errors using sklearn_metrics 
def metric(validation_set_status,pred):
    print('Mean Absolute Error:', metrics.mean_absolute_error(validation_set_status,pred))
    print('Mean Squared Error:', metrics.mean_squared_error(validation_set_status,pred))
    print('Root Mean Squared Error:', np.sqrt(metrics.mean_squared_error(validation_set_status,pred)))
